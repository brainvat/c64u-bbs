*** c64u-bbs ***

Commodore 64 Unicode BBS █

About

c64u-bbs lets you deploy and manage a Commodore 64 BBS on the C64 Ultimate — entirely over the network. Clone the repo, run the installer, and talk to your C64U from the command line. No SD cards, no manual file transfers.

Features

  • c64u CLI tool for managing your C64 Ultimate remotely
  • REST API client with X-Password authentication
  • One-command install with bash install.sh
  • Git hooks scanning every commit for leaked secrets
  • Drive management, PRG runner, FTP file access (coming soon)
  • Automated BBS deployment via ACIA modem emulation (coming soon)

Getting Started

READY.
git clone https://github.com/brainvat/c64u-bbs.git
cd c64u-bbs && bash install.sh
source venv/bin/activate
c64u --host <your-c64u-ip> info

See the GitHub repository for full setup instructions.
